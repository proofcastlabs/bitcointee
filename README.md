# Bitcointee

## Requirements

 - Android SDK 33+
 - Android NDK 25.2
 - Rust installed
 - `armv7-linux-androideabi` rust toolchain

## Project setup

### `local.properties`

```
sdk.dir=/opt/android-sdk
ndk.dir=/opt/android-sdk/ndk/25.2.9519653
rust.rustcCommand=/<home>/.cargo/bin/rustc
rust.cargoCommand=/<home>/.cargo/bin/cargo
```
